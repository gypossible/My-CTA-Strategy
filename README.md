# My CTA Strategy (中国大陆商品期货量化回测与模拟实盘框架)

## 简介
这是一个支持**中国大陆商品期货市场 (Commodity Futures)** 的量化交易、极值评测与实盘模拟测试框架。
项目目前已迭代至 V3 版本，不仅内置了极速的 Pandas 向量化回测引擎进行毫秒级数据跑批，还抽象出了实盘接口层，方便投研团队一键验证并无缝进入实战（Paper Trading）。

## 海龟交易法则 (Turtle Trading System 1) 核心操作细节

量化界最负盛名的“海龟法则”的核心哲学是建立在唐奇安通道（Donchian Channel）上的绝对趋势跟踪体系。本策略库内置并执行的是最为严苛也是最为知名的**系统一 (System 1)**，操作细则完全解密如下：

- **入场逻辑（开单）：**
  - **做多进场 (Long Entry)**：当某一交易日的收盘价，向上突破了**过去 20 个交易日的最高价**时，说明猛烈的上行趋势已经确立，立即建立多头头寸（买开）。
  - **做空进场 (Short Entry)**：当某一交易日的收盘价，向下跌破了**过去 20 个交易日的最低价**时，说明下行深渊开启，市场崩盘，立即建立空头头寸（卖开）。
- **离场逻辑（止盈 / 止损）：**
  - **做多出场 (Long Exit)**：持有多单时，一旦价格反转向下跌破**过去 10 个交易日的最低点**，多头无条件斩仓平仓离场。
  - **做空出场 (Short Exit)**：持有空单时，一旦价格反弹向上突破**过去 10 个交易日的最高点**，空头无条件止损平仓离场。

> **风控警告：**标准的金融海龟体系强制配合基于 ATR（Average True Range，平均真实波动幅度）的头寸规模管理（Position Sizing）进行动态加减仓（即：波动越大，买的越少）。本框架的信号层已完美实现海龟的“开平仓通道逻辑”，但在回测执行时预设使用系统单一杠杆。如果您使用 `trader_api.py` 接入实盘，务必自行加入 ATR 手数折算逻辑控制爆仓风险！

## 架构模块说明
- `strategy.py`: 存放所有 CTA 相关信号发生器引擎（包含海龟突破、双均线趋势、震荡 RSI 均值回归）。
- `trader_api.py`: 独立的 `SimulatedPaperTrader` 纸面模拟收单引擎。可接驳外部 CTP。
- `metrics.py`: 十大严苛评判矩阵引擎（包含 Calmar 卡玛比率、Sortino 索提诺比率、Profit Factor 盈亏比等指标的计算）。
- `main.py`: 项目入口点，一键拉取 2026 年最新 API 极速并行生成 21 组平行回测对比。
- `templates/` 和 `reporter.py`: 解析生成具备暗夜模式的高能全中文金融大屏 HTML。

## 快速启动

2. 启动引擎：`python main.py`
3. 阅览报告：`reports/backtest_report.html` 自动生成。
